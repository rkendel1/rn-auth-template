# Features

* React Native
* React Navigation
* Async Storage with [React Native CryptoJS](https://github.com/imchintan/react-native-crypto-js)
* axios
* Support UI Kitten theme

```
#ncu
#ncu -u
yarn
yarn web
expo publish
```
